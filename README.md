## To-do App

### Funcionalidades:
- [x] Criar, deletar ou editar um to-do;
- [x] Marcar como completo ou imcompleto;
- [x] Todos os to-do's são salvos no localstorage, permitindo o usuário dar reload ou sair e voltar da página.
